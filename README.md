# 🎯 LGS Performans Analizi

Özel Gaziantep Bilfen Okulları 8. Sınıf LGS Hazırlık Öğrenci Performans Takip Sistemi

## 📊 Canlı Demo

👉 **[Sistemi Görüntüle](https://kursatovic.github.io/lgs-analiz/)**

## ✨ Özellikler

- 📈 **21 Sınav** takibi (tüm dersler)
- 🎯 **Sınıf Filtresi** (8-A / 8-B)
- 💪 **Gelişim Fırsatı Filtresi** (en zayıf ders bazlı)
- 👤 **Öğrenci Profilleri** (sınav bazlı net kartları)
- 📊 **İnteraktif Grafikler** (tıklanabilir trend analizi)
- 🎨 **Görsel Raporlar** (yüzdelik başarı gösterimi)

## 🚀 Kullanım

1. Sisteme girin
2. Sınıf seçin (8-A / 8-B / Tümü)
3. İsteğe bağlı: Gelişim fırsatı filtresi uygulayın
4. Öğrenci seçin
5. **"📊 Profili Görüntüle"** butonuna basın
6. 21 sınavın detaylarını görün

## 🎨 Profil Kartları

Her ders için:
- 🟢 **Yeşil kart** = Ortalamanın üstü
- 🔴 **Kırmızı kart** = Ortalamanın altı
- 🟢 **Yeşil çerçeve** = En yüksek sınav
- 🔴 **Kırmızı çerçeve** = En düşük sınav
- ⬜ **Gri "G"** = Sınava girmedi
- **↗ / ↘ / →** = Trend göstergesi
